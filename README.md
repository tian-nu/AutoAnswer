# AutoAnswer - 自动答题助手

智能识别网页题目，调用 AI API 自动作答。支持超星学习通平台。

## 功能特性

- 🔍 **智能检测**：自动识别页面中的选择题、填空题、判断题、简答题
- 🤖 **AI 答题**：调用 OpenAI / DeepSeek / 自定义 API，自动获取答案并填写
- 🎯 **平台适配**：超星学习通（chaoxing.com）
- 📊 **实时统计**：检测数、成功数、失败数一目了然
- 🖱️ **自由交互**：点击标题栏展开/收起，拖动移动，右下角调整大小
- 💾 **状态持久化**：悬浮窗位置、尺寸、折叠状态自动保存

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 [安装脚本](https://github.com/tian-nu/AutoAnswer/raw/main/%E8%87%AA%E5%8A%A8%E7%AD%94%E9%A2%98.user.js)
3. Tampermonkey 会自动识别并安装

## 使用

1. 打开超星学习通的答题页面
2. 页面右上角出现 **"答"** 胶囊按钮
3. 点击展开悬浮窗
4. 在 **API** 标签页配置 API Key 和模型
5. 点击 **"开始答题"** 即可

## 配置说明

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| API Key | AI 服务的密钥 | 空 |
| 模型 | 使用的模型名称 | gpt-4o-mini |
| Base URL | API 基础地址 | https://api.openai.com |
| 并发数 | 同时答题数量 | 3 |
| 答题延迟 | 每题间隔时间(ms) | 1000 |
| 自动检测 | 页面加载后自动识别题目 | 开启 |
| 自动填写 | 获取答案后自动填入 | 开启 |

### 支持的 API 提供商

- **OpenAI** — GPT-4o-mini / GPT-4o / GPT-3.5-turbo
- **DeepSeek** — deepseek-chat / deepseek-reasoner
- **自定义** — 任意兼容 OpenAI 格式的 API

## 支持的题目类型

| 多选题 | ✅ |
